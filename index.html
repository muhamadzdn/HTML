Apa itu HTML ?

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML digunakan untuk menyusun konten di web, termasuk teks, gambar, tautan, dan elemen lainnya. Dengan HTML, pengembang web dapat menentukan struktur dan tata letak halaman web.

Heading

Tag heading digunakan untuk membuat judul dan subjudul dalam dokumen HTML. Tag heading memiliki enam tingkat, dari <h1> hingga <h6>, dengan <h1> sebagai judul utama dan <h6> sebagai subjudul terkecil.

  • <h1> - Judul utama
  • <h2> - Subjudul tingkat 1
  • <h3> - Subjudul tingkat 2
  • <h4> - Subjudul tingkat 3
  • <h5> - Subjudul tingkat 4
  • <h6> - Subjudul tingkat 5

Paragraph

Paragraf adalah blok teks yang digunakan untuk menyusun konten dalam dokumen HTML. Tag <p> digunakan untuk membuat paragraf, dan biasanya digunakan untuk menulis teks yang panjang atau menjelaskan suatu topik. Paragraf membantu memisahkan ide-ide dan membuat konten lebih mudah dibaca.

Reserved Elements

Reserved elements adalah elemen HTML yang memiliki makna khusus dan tidak dapat digunakan sebagai nama elemen kustom. Contoh elemen reserved termasuk <div>, <span>, <header>, <footer>, dan <section>. Elemen-elemen ini digunakan untuk menyusun dan mengatur konten dalam dokumen HTML.

  • <div> - Elemen blok untuk mengelompokkan konten
  • <span> - Elemen inline untuk menandai teks
  • <header> - Elemen untuk menandai bagian header dari dokumen atau bagian

Void Elements

Void elements adalah elemen HTML yang tidak memiliki tag penutup. Contoh void elements termasuk <img>, <input>, <br>, dan <hr>. Elemen-elemen ini digunakan untuk menyisipkan konten atau memisahkan bagian dalam dokumen HTML.

  • <div> - Elemen blok untuk mengelompokkan konten
  • <span> - Elemen inline untuk menandai teks
  • <header> - Elemen untuk menandai bagian header dari dokumen atau bagian

Style Elements

Style elements adalah elemen HTML yang digunakan untuk menambahkan gaya atau format pada konten. Contoh style elements termasuk <strong>, <em>, <u>, dan <mark>. Elemen-elemen ini digunakan untuk menekankan teks, memberikan penekanan visual, atau menyoroti bagian tertentu dari konten.

  • <div> - Elemen blok untuk mengelompokkan konten
  • <span> - Elemen inline untuk menandai teks
  • <header> - Elemen untuk menandai bagian header dari dokumen atau bagian
